# LeetGFGHub

> Automatically sync LeetCode and GeeksForGeeks solutions to GitHub.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg?style=flat)](https://github.com/abhinay1376/leetgfghub/issues)

LeetGFGHub is a Chrome Extension that automatically pushes your **accepted** LeetCode and GeeksForGeeks solutions to GitHub, generates per-problem READMEs, and tracks your progress with a live analytics dashboard.

## ✨ Features

- **Accepted-only sync** — triggers only on new accepted submissions, never on WA, TLE, or history views
- **Smart organization** — structured folder per problem with auto-generated README
- **Live analytics** — streaks, difficulty breakdown, language stats rebuilt from repository
- **GitHub as source of truth** — analytics stored in `.dsa-sync/`, fully restorable
- **Premium UI** — Apple-inspired glassmorphism panel, dark mode

## 🚀 Installation

1. Clone: `git clone https://github.com/abhinay1376/leetgfghub.git`
2. Open `chrome://extensions/` → Enable **Developer mode** → **Load unpacked** → select the folder.

*(Coming soon to the Chrome Web Store)*

## ⚙️ Setup

1. Create a GitHub repo and generate a [Personal Access Token](https://github.com/settings/tokens) with `repo` scope.
2. Open the extension → enter your token and repository URLs → **Verify Connection**.

## 🔄 Restore from Repository

Re-installed or changed computers? Go to **Settings → Repository Sync → Restore**. LeetGFGHub will rebuild your full history from the repo.
