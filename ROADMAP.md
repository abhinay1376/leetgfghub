# 🗺️ LeetGFGHub Roadmap

This document outlines the planned features and technical goals for LeetGFGHub.

## 🚀 Near Term (Q3)

- **Chrome Web Store Publish:** Prepare all promotional assets and undergo Chrome Web Store review.
- **HackerRank Support:** Add support for syncing HackerRank solutions.
- **Customizable Commit Messages:** Allow users to define their own commit message templates.
- **Keyboard Shortcuts:** Add hotkeys for quick access to the popup and syncing manual changes.

## 🌟 Medium Term (Q4)

- **Codeforces Support:** Integrate with Codeforces for competitive programming tracking.
- **Multi-File Sync:** Better support for problems that require multiple files or classes.
- **Local Git Sync:** Explore optional desktop-agent integration for users who prefer syncing directly to a local `.git` directory rather than via the API.

## 🏗️ Long Term

- **Social Features:** Compare streaks and progress with friends via the dashboard.
- **AI Code Review:** Optional integration to provide small performance or readability tips on submitted solutions.

*Note: This roadmap is subject to change based on community feedback and contributions.*
