# Security Policy

## Supported Versions

Currently, only the latest version of LeetGFGHub is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within LeetGFGHub, please do not disclose it publicly.

Instead, please send an e-mail to the maintainers or open a private security advisory on GitHub.

We will review the issue and, if verified, release a patch as quickly as possible. Please allow up to 48 hours for an initial response.

**Important Note regarding GitHub Tokens:**
LeetGFGHub requires a GitHub Personal Access Token (PAT) to commit code to your repositories. This token is stored securely in your browser's local `chrome.storage.sync`. It is never transmitted to any third-party servers; it is only used to communicate directly with the GitHub API (`api.github.com`).
